# NearbyHelper 🛠️

**NearbyHelper** is a web application designed to connect users with local service providers such as plumbers, electricians, carpenters, and other professionals. This project aims to simplify finding reliable workers nearby and improve visibility for skilled professionals.

---

## 🚀 Features

- 🧑‍💼 **User & Worker Registration/Login**
  - Separate sign-up forms for users and service providers
  - Workers provide additional info: skill, phone, address, profile photo

- 🛠️ **List of Workers**
  - Users can view all registered service providers
  - Workers can see their own profiles and others in the community

- 🔎 **Skill-Based Filtering (Upcoming)**
  - Filter workers by profession (e.g., plumber, electrician, etc.)

- 📱 **Responsive UI**
  - User-friendly design built with React.js and Bootstrap
